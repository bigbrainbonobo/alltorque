## Recommended Visual Style

### Color Palette
**Primary:**
- Deep industrial charcoal (#1a1a1a - #2d2d2d)
- Diesel black (#0a0a0a)

**Accent:**
- High-visibility orange/amber (#ff6b00 - #ff8c00) - industrial warning color
- OR Torque red (#c41e3a) - power/strength
- OR Steel blue (#4a7c9e) - industrial/mechanical

**Supporting:**
- Concrete gray (#808080 - #a0a0a0)
- Bright white (#ffffff) - for contrast/readability
- Warm off-white (#f5f5f0) - for backgrounds

**Approach:**
- High contrast (dark backgrounds, bright text)
- Industrial color psychology
- Accent color used sparingly for CTAs and emphasis
- Avoid pastels, soft colors, consumer-friendly palettes

---

### Typography
**Headlines:**
- Bold, strong, industrial feel
- Sans-serif, heavy weights (700-900)
- Suggested: Inter Black, Montserrat ExtraBold, Work Sans Bold, Bebas Neue
- Large scale (48px-72px desktop, 32px-48px mobile)
- High contrast against backgrounds

**Body:**
- Clean, readable sans-serif
- Medium weight (400-500)
- Suggested: Inter, Work Sans, Source Sans 3
- Size: 16px-18px base
- Line height: 1.6-1.8 for readability

**Accents:**
- Uppercase for labels/categories
- Letter spacing on uppercase (+0.05em)
- Numerical data: tabular figures, bold weights

**Hierarchy:**
- Strong size differentiation
- Weight contrast
- Color used for emphasis (accent on CTAs)

---

### Layout & Spacing
**Grid:**
- 12-column grid (desktop)
- Generous gutters (32px-48px)
- Breakpoints: 1440px, 1024px, 768px, 375px

**Spacing:**
- Large section padding (120px-160px desktop, 60px-80px mobile)
- Breathing room around key elements
- Asymmetric layouts for visual interest

**Composition:**
- Bold full-width sections alternating with contained content
- Split-screen layouts (50/50 image/content)
- Diagonal cuts/angles for industrial feel
- Overlapping elements for depth

---

### Textures & Effects
**Backgrounds:**
- Subtle industrial textures (concrete, metal, diagonal patterns)
- Dark gradients (charcoal to black)
- Overlay effects on images (dark wash for text readability)

**Effects:**
- Sharp shadows for depth (not soft/blurry)
- Subtle noise/grain for industrial texture
- High-contrast borders
- Angled/diagonal elements

**Cards/Components:**
- Sharp corners OR slight radius (4px-8px, not heavily rounded)
- Depth through shadows and borders
- Hover states with lift/elevation

---

### Motion Language
**Philosophy:**
- Intentional, not gratuitous
- Enhances hierarchy and flow
- Industrial/mechanical feel (smooth but powerful)

**Timing:**
- Duration: 0.4s-0.8s for major movements
- Easing: ease-out for entrances, ease-in-out for interactions
- Stagger: 0.1s-0.15s between sequential elements

**Scroll Animations:**
- Fade + slide up (60px-100px)
- Stagger on grids/lists
- Parallax on hero backgrounds (subtle, 20-30% speed difference)
- Sticky navigation with backdrop

**Hover/Interaction:**
- Lift effect on cards (translateY -8px, shadow increase)
- Color transitions on CTAs (0.3s)
- Icon micro-animations
- Border/underline effects on links

**Hero:**
- Text stagger reveal (0.6s, 0.15s stagger)
- Background zoom (subtle, 1.05x scale on load)
- Parallax on scroll
- CTA pulse/glow (subtle)

**Avoid:**
- Bouncing animations
- Spinning/rotating (unless mechanical-themed)
- Excessive duration (>1s)
- Animations that block content
- Motion that distracts from conversion

## Implementation Notes

### Technical Stack Recommendations
- Vanilla HTML/CSS/JavaScript (no framework overhead)
- CSS Grid + Flexbox for layouts
- CSS custom properties for theming
- Intersection Observer API for scroll animations
- CSS transforms for motion (GPU-accelerated)
- Lazy loading for images
- Next-gen image formats with fallbacks

### Performance Priorities
- First Contentful Paint < 1.5s
- Largest Contentful Paint < 2.5s
- Time to Interactive < 3.5s
- Mobile-optimized imagery
- Minimal JavaScript payload
- Critical CSS inline
- Defer non-critical assets

### Accessibility
- WCAG 2.1 AA compliance minimum
- Keyboard navigation
- Screen reader optimization
- Sufficient color contrast (especially on dark backgrounds)
- Focus indicators
- Reduced motion media query (disable animations for users with motion sensitivity)
- Alt text for all images
- Semantic HTML

### Mobile Considerations
- Touch-friendly CTAs (minimum 44px tap target)
- Sticky phone button (bottom-right, always accessible)
- Simplified navigation (hamburger menu)
- Optimized image sizes for mobile bandwidth
- Form inputs optimized for mobile keyboards
- Horizontal scrolling avoided
- Single-column layouts below 768px
